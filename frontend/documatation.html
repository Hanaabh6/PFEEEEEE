Guide d’utilisation IntelliBuild

Comment naviguer dans votre bâtiment intelligent. Cette page vous accompagne dans la découverte de la plateforme pour localiser, interagir et gérer vos ressources connectées au sein du bâtiment.

Commencer Retour accueil

Ce guide vous montre

  • Comment démarrer et accéder à votre espace personnel.
  • Comment rechercher et localiser rapidement un objet.
  • Comment interagir avec les objets connectés.
  • Comment suivre votre historique et vos notifications.
  • Le workflow idéal pour une utilisation simple et efficace.

1. Démarrage rapide

Pour commencer, créez un compte ou connectez-vous. L'application vous dirigera vers votre Tableau de Bord personnel.

1

Inscription

Créez votre profil en quelques secondes.

2

Connexion

Accédez à votre espace sécurisé.

3

Langue

Adaptez l'interface selon votre préférence via le sélecteur en haut de page.

4

Navigation

Une fois connecté, explorez les pages utiles depuis votre espace personnel.

2. Rechercher et Localiser

Le cœur d'IntelliBuild est son moteur de recherche intelligent :

  • Recherche sémantique : Tapez simplement le nom ou le type d'un objet (ex: "Capteur", "Imprimante").
  • Localisation 3D : Utilisez la page Localisations pour visualiser l'architecture du bâtiment et sélectionner une zone précise pour filtrer les objets à proximité.
  • État initial : Par défaut, la liste est vide pour vous permettre de lancer une recherche ciblée.

3. Interagir avec les objets

Une fois l'objet trouvé, vous disposez de plusieurs outils pour l'exploiter :

  • Contrôle en direct : Interagissez avec les objets compatibles (allumer/éteindre, consulter les mesures).
  • Favoris : Ajoutez vos objets fréquents à votre liste de Favoris
  • Signalement : Si un objet ne fonctionne pas correctement, utilisez le bouton "Signaler un problème" pour avertir l'administration.

4. Suivi et Historique

Gardez une trace de toutes vos activités dans le bâtiment :

  • Mon Historique : Consultez la liste de vos interactions passées avec les objets.
  • Notifications : Recevez des alertes importantes (disponibilité d'une ressource, maintenance, etc.).
  • Profil : Gérez vos informations personnelles et vos préférences de sécurité.

5. Workflow idéal (Utilisateur)

A

Recherchez l'objet souhaité via la barre de recherche.

B

Se situer

Vérifiez la position de l'objet sur le plan du bâtiment.

C

Utiliser

Interagissez avec l'objet selon vos besoins.

D

Signaler

En cas de panne, envoyez un signalement immédiat.

FAQ (Questions fréquentes)

Comment retrouver mes objets préférés ?

Ils sont regroupés dans l'onglet "Favoris" de votre espace personnel.

Pourquoi je ne vois aucun résultat ?

Assurez-vous d'avoir saisi un mot-clé ou vérifiez que l'objet n'est pas hors-ligne.

Que faire si un objet est en panne ?

Cliquez sur l'objet, puis sur l'icône de signalement pour créer une alerte automatique.

Comment naviguer dans le bâtiment ?

Utilisez la page Localisations et la barre de recherche pour filtrer les objets proches.
